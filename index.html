TOBJ Creator is a Python program that makes TOBJ files for ETS2, ATS and most likely every other SCS game.

Running

Windows
  1. Download Python 3.7 and install it if you haven't already.
  2. Download the latest version above and extract the downloaded zip file.
  3. Double click the tobjcreator.py file inside the src folder.

Linux
  1. Install Python 3.7 using sudo apt install python3 in the console if you haven't already.
  2. Download the latest version above and extract the downloaded zip file.
  3. Navigate to the extracted folder in the console and type python3 src/tobjcreator.py.

Usage

To use the program you will need a base TOBJ file. What that means is that you have to find a TOBJ file created by SCS, e.g. if you're making a recolor of anti_noise6 you can use anti_noise6.tobj as a base. The rest should be pretty self-explanatory.
